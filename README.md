# Repositórios Favoritos

### Projeto consiste em favoritar os repositorios mais utilizados por você, assim obtendo uma consulta mais rapida ao repositório.

## Tecnologias usadas:
- React JS
- Hooks
 - - useState
, useEffect, useCallback
- Axios
- Styled-Components
- Local Storage
